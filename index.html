Quick Reference: Chomsky Hierarchy

Type Grammar Language Automaton Production Rule Form
Type 0 Unrestricted Recursively Enumerable Turing Machine alpha -> beta (no restrictions)
Type 1 Context-Sensitive Context-Sensitive Linear Bounded Automata alpha A beta -> alpha gamma beta
Type 2 Context-Free Context-Free Pushdown Automata A -> gamma
Type 3 Regular Regular Finite Automata A -> aB or A -> a

Note: Type 3 is a subset of Type 2, Type 2 is a subset of Type 1, and Type 1 is a subset of Type 0.

Important Symbols and Notation

Symbol Meaning Example
Sigma Alphabet (finite set of symbols) {0, 1}, {a, b, c}
Sigma* Kleene closure (all strings over alphabet) {epsilon, 0, 1, 00, 01, 10, 11, ...}
Sigma+ Positive closure (all non-empty strings) {0, 1, 00, 01, 10, 11, ...}
epsilon Empty string (length 0) |epsilon| = 0
|w| Length of string w |abc| = 3
delta Transition function delta(q0, a) = q1
delta* Extended transition function delta*(q0, ab) = q2
L(M) Language accepted by machine M L(M) = {w : delta*(q0, w) in F}
